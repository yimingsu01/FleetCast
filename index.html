FleetCast Helm Repository

This GitHub Pages site hosts the Helm chart for FleetCast, a Satellite Operations Simulator.

Usage

helm repo add fleetcast https://yimingsu01.github.io/FleetCast
helm install satellite-app fleetcast/satellite-app -f values.secret.yaml
